# tjv-projekt

## Blog Social Network

Uživatelé budou moci vytvářet vlastní blogy a na nich nové příspevky.
Dále mohou "lajkovat" jednotlivé příspěvky
a také přidávat příspěvky do svých blogů.

Projekt bude používat relační databázi.
